Example

The best way to the use widget is to use an iframe to embed it onto your site.

<iframe src="http://curse.modmuss50.me/widget/233564" width="800" height="150" scrolling="no" seamless="seamless" style="border: none;"></iframe>

Remember that the first time a project is requested it will take slightly longer to load while the data is retrieved from CurseMeta. Project data is cached for 30 mins.

The downloader counter will count up, simulating a live download counter. This is NOT a live download counter, but is based on the last 30 days of downloads on the project and then averaged out. This can be disabled by appending the following to the url.

?simulateDownloadCount=false
